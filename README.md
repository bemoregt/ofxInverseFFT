# ofxInverseFFT
inverse 2d FFT using Openframeworks C++ in MacOSX

![ifft example]( https://github.com/bemoregt/ofx2DFFT/blob/master/https://github.com/bemoregt/ofxInverseFFT/blob/master/test.png "ifft")

### Dependency
- OpenFrameworks 0.10.1
- ofxCv
- ofxOpenCv
- ofxGUI
- XCode 10.12.x
- OSX Mojave
